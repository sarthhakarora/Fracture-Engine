# Fracture Engine

Fracture Engine is a minimal, gameplay-focused engine built in C using raylib.

It is derived from Lattic Engine, but heavily stripped down to remove UI, tooling, and simulation-specific systems. The goal is to provide a lightweight foundation for building fast-paced games with minimal overhead.

## Goals

- Minimal abstraction over raylib
- Clean and simple game loop
- Easy to modify and extend
- Focus on gameplay, not tooling

## Status

Early development. This is a stripped base and will evolve alongside the game it is built for.

## Tech

- C (C11)
- raylib
- raymath

## Note

This is not a general-purpose engine. It is being built specifically to support a game project and will grow based on real needs.
